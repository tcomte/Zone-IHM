    En savoir plus sur Reador!

    Se tenir informer

    Suivez en temps réel les news publiées sur internet avec un filtrage personnalisé des informations.

    Connectez moi à twitter!

    Reador peut récupérer votre timeline et effectuer du filtrage pour vous sur les tweets présents en ligne.

    Scalable

    Reador marche aussi bien sur PC que sur tablette ou telephone portable!

    OpenSource

    Reador est un logiciel libre, vous pouvez le telecharger, le modifier et le publier, effectuez des pull requests sur github.

    Vous pouvez générer vos propre flux RSS sur Reador pour ensuite les utiliser sur votre propre gestionnaire de flux RSS

    Reador est gratuit

    Reador est fournit gratuitement depuis le site web reador.net.

    Fait pour vous et avec vous!

    Reador est développé pour vous! Nous avons réelement besoin de vos retours, de vos besoin et de tout savoir sur l'organisation de votre veille pour vous fournir une solution toujours plus performante soumettre une requête.

    Semantique

    La sémantique est un point qui nous tient vraiment à coeur, nous voulons supprimer toutes sources d'ambiguité dans vos recherches de news.

    Une solution parfaite pour les veilleurs

    Vous êtes stressés à l'idée de rater une news importante? d'avoir du retard sur vos news à analyser? Nous aussi!

    Qui est dérrière reador?

    Reador est principalement développé par Christophe Desclaux. Il est assisté par l'institut de recherche Inria dans le cadre du concours BoostYourCode2012.
